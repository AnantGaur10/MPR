Quick Start Guide: ASL to Audio

Follow these steps to get the application running correctly. This guide can be accessed anytime by clicking the "Help" button in the app.

Step 1: Start the Backend Server (Developer Step)

Before using the app, the Go backend server must be running. This process is for developers only.

  1. Open a terminal or command prompt.
  2. Navigate to the ml-service-backend project folder.
  3. Run the command: go run .

Step 2: Set Up OBS Studio (One-Time User Setup)

This application requires the "OBS Virtual Camera" to share your webcam with multiple programs. You only need to do this setup once.

  1. Open OBS Studio (the installer should have placed a shortcut on your desktop).
  2. In the "Sources" panel at the bottom, click the + button and add a Video Capture Device.
  3. In the properties window that opens, select your physical webcam from the "Device" dropdown and click OK.
  4. Resize the webcam preview in the main OBS window to fill the black screen.
  5. In the "Controls" panel on the right, click Start Virtual Camera.
  6. You can now minimize the OBS window. It will continue running in the background.

Step 3: Run the ASL to Audio App

  1. Launch the "ASL to Audio" application from your desktop or Start Menu.
  2. The "Camera Source" dropdown should automatically select OBS Virtual Camera. If not, select it manually.
  3. Click the large power button to start the service. The status should change to "Connected".

Step 4: Configure Your Meeting App (e.g., Google Meet)

  1. Join your video call in Google Meet, Zoom, Discord, etc.
  2. Go to the audio and video settings within that meeting application.
  3. Set your Camera to OBS Virtual Camera.
  4. Set your Microphone to CABLE Output (VB-Audio Virtual Cable).

Troubleshooting

  • "OBS Cam not found" error in the app: Make sure OBS is running and you have clicked "Start Virtual Camera".
  • "VB-CABLE not found" error in the app: This means the driver did not install correctly. Please re-run the application installer that you downloaded.
  • No audio in the meeting for others: Ensure "CABLE Output" is selected as the microphone in your meeting app.
  • Can't hear others in the meeting: Make sure your real headphones/speakers are set as the "Speakers" or "Audio Output" in your meeting app's settings.
